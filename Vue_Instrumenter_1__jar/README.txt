
How to use:

Open with WINDOWS cmd:
To instrument:
java -jar Vue_Instrumenter_1_.jar -i <..path..>
To deinstrument:
java -jar Vue_Instrumenter_1_.jar -d <..path..>

Example: 
java -jar Vue_Instrumenter_1_.jar -i C:\Users\ASUS\WebstormProjects\stackoverflow_vue\src


To mark a Vue file NOT to be instrumented: add /*no-instru*/ anywhere in that file.
